# sketch_nano
sketch nano desu

## 手元の環境に持ってくる（cloneする）
`git clone https://github.com/fsubal/sketch_nano.git`

## 使用Gemパッケージ
* sinatra
* sqlite3
* sinatra-contrib
* data_uri
* securerandom (requireはいるけどgem installはいらない)

`gem install data_uri` のようなコマンドを打つ。

失敗したら `sudo gem install data_uri` -> 自分のPCのパスワード

## 起動するときは

```sh
cd sketch_nano
ruby main.rb
```
